# React Notes App

## Available commands

Run in development mode

```bash
  npm run dev
```

Create production build

```bash
  npm run build
```

Run ESLint linting

```bash
  npm run lint
```

Run Prettier formatting

```bash
  npm run format
```

Run TypeScript compiling

```bash
  npm run compile
```

Serve production build locally

```bash
  npm run preview
```
